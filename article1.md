# 生物信息学 – 第1讲课堂笔记

## 一、什么是生物信息学？

**Bioinformatics = Biology + Data Science + Computation**  

生物信息学是在大规模生物数据（基因组学、转录组学、蛋白组学等）背景下，利用计算方法对生命系统进行分析、建模和解释。  

**核心问题：**

1. 数据如何存储？  
2. 数据如何处理？  
3. 如何将“杂乱”的数据转化为“有意义”的信息？

---

## 二、现代生物数据特点（Big Data in Biology）

**现代生命科学特征：**

* 高通量测序（NGS）  
* 单细胞测序  
* 多组学整合（Multi-omics）  
* 大规模临床与分子数据整合  

**数据四大特征：**

| 特征 | 说明 |
| ---- | ---- |
| Volume | 数据量巨大 |
| Variety | 数据类型多样 |
| Velocity | 数据更新快 |
| Noise | 数据噪声复杂 |

**结论：**  
> 生物科学正从“假设驱动”逐步走向“数据驱动”。

## 三、Algorithm 与 Model 的区别（重点）

这是本节课最核心的概念，直接影响后续所有生信分析。

### 1️⃣ Algorithm（算法）

**定义：**  
一套明确、有限、可执行的步骤，用于解决具体问题。  

**特点：**

* 有输入与输出  
* 关注“如何计算”  
* 强调执行流程  
* 可以直接编程实现  

**例子：**

* 动态规划（序列比对）  
* K-means 聚类  
* 排序算法  
* 最短路径算法  

**一句话理解：**  
> Algorithm 是“解决问题的方法步骤”。

---

### 2️⃣ Model（模型）

**定义：**  
对现实世界的一种抽象表达，用数学结构刻画系统规律。  

**特点：**

* 关注“系统是什么”  
* 包含假设  
* 可解释或可预测  
* 通常含有参数  

**例子：**

* 线性回归模型  
* 逻辑回归模型  
* 隐马尔可夫模型（HMM）  
* 神经网络模型  

**一句话理解：**  
> Model 是“对现实的抽象假设”。

---

### 3️⃣ 二者关系对比

| 维度        | Algorithm | Model |
| --------- | -------- | ----- |
| 本质        | 计算步骤 | 抽象表达 |
| 关注点      | 怎么算   | 怎么描述 |
| 是否含假设  | 不一定   | 必须包含 |
| 举例        | 最小二乘法 | 线性回归 |

**核心理解：**  

* Model 决定结构  
* Algorithm 决定求解方式  

**例子说明：**  

* **线性回归** 是模型 → 描述输入与输出之间的关系  
* **最小二乘法** 是算法 → 求解该模型的参数

## 四、高维数据（High Dimension）

**特点：**

* 一个样本可能包含上万个基因表达值  
* 特征数远大于样本数  

**带来的问题：**

* 维度灾难（Curse of dimensionality）  
* 容易过拟合  
* 距离度量失效（高维空间中“距离”意义减弱）  

**解决思路：**

* 降维（PCA）  
* 正则化（L1 / L2）  
* 特征选择

## 五、本学期学习计划

### 总目标

* 理解核心模型的数学本质  
* 掌握基础算法思想  
* 能够分析基础生物数据  

### 第一阶段（1–4周）：基础能力建设

* 复习线性代数（矩阵运算、特征值）  
* 复习概率论（分布、期望、方差）  
* Python 科学计算（numpy, pandas）  

**重点：**

* 深刻理解“模型与算法的区别”  
* 理解假设在建模中的作用

### 第二阶段（5–10周）：核心方法学习

* 线性回归与逻辑回归  
* 聚类算法（K-means）  
* PCA 降维  
* 基础序列比对思想  

**实践目标：**

* 推导核心公式  
* 手写简化算法  
* 分析小规模真实数据

### 第三阶段（11–16周）：综合提升

* 神经网络基础  
* 多组学整合思路  
* 阅读 2–3 篇生信论文  
* 完成一个小型数据分析项目  

**目标：**

* 从“会使用工具”提升到“理解原理”  
* 从“做计算”提升到“做解释”

## 六、本节课核心收获

1. 生物信息学本质上是数据科学  
2. Model 是假设与结构  
3. Algorithm 是求解路径  
4. 高维数据是现代生命科学的核心挑战

---

> 未来科研能力不仅是实验能力，  
> 还包括建模能力与算法理解能力。
